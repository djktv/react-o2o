# react-o2o

> react 16+react-router 4+redux 仿大众点评

## Build Setup

``` bash
# install dependencies
yarn

# mock serve with hot reload at localhost:8080
yarn mock

# serve with hot reload at localhost:3000
yarn start

# build for production with minification
yarn build

```
